# Oceanic Next _Italic_ for Atom

## Installation

### Atom Package Manager (APM)
```bash
apm install atom-oceanic-next-italic
```

### Git clone
```bash
cd ~/.atom/packages
git clone https://github.com/AndrewKishino/atom-oceanic-next-italic --depth=1
```
